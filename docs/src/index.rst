SatOS Payload SDK
=================

The SDK documentation contains a number of useful resources:

* A comprehensive :doc:`Developer Guide </developer-guide>` helps developers with varying degrees of experience work effectively with the SDK
* The :doc:`Payload Interface API Reference </payload-interface>` describes at an API-level how payloads communicate with core services 

The source for this documentation and the SDK itself is available on Github: https://github.com/antaris-inc/SatOS-Payload-SDK/.
